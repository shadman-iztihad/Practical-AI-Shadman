API Documentation

Overview

This API allows you to analyze images using Azure's Computer Vision OCR service. You can extract text from images by sending a properly formatted request to the API.

API Endpoint

The API is accessible at: http://localhost:3000/api/v1/analysis/

Request

To analyze an image, send a POST request with a JSON object containing a single field "uri", which should point to the image URL to be analyzed. 

            
{
    "uri": "http://jeroen.github.io/images/testocr.png"
}

Response

The API will return a JSON object with the extracted text or an error message. Example response: 

            
{
    "text": "Extracted text from the image."
}

Error Handling

In case of an error, the API will return an error message in JSON format. Example: 

            
{
    "error": "The provided URI is not a valid URL."
}

Common errors include:

  • 400: Missing or invalid uri field in the request.
  • 500: Internal server error during processing.
  • 408: OCR process took too long to complete.
